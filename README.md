## 📌 Descrição

Este repositório contém as **atividades, exercícios e projetos desenvolvidos na disciplina de Desenvolvimento de Dispositivos Móveis.** O objetivo é aplicar conceitos de programação mobile, criação de interfaces e lógica de aplicativos.

## 🎯 Objetivo

Desenvolver habilidades na criação de aplicativos móveis, utilizando ferramentas e tecnologias voltadas para o desenvolvimento mobile, além de compreender conceitos como **interface, usabilidade e lógica de programação.**
