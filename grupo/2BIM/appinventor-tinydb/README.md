# 💾 Pesquisa - App Inventor: Manipulação de Banco de Dados no Dispositivo (TinyDB)

Pesquisa acadêmica desenvolvida sobre os conceitos fundamentais, funcionamento e boas práticas no uso do componente TinyDB para persistência de dados local no MIT App Inventor.

---

## 📌 Sobre o Projeto

Este trabalho apresenta um estudo detalhado sobre o armazenamento de dados em dispositivos móveis utilizando o ecossistema do MIT App Inventor. O foco principal está no componente não visível TinyDB, explorando seu mecanismo de funcionamento baseado na estrutura Chave-Valor (Tags e Valores).
Também são analisadas as diferenças arquiteturais entre soluções de armazenamento local e em nuvem (TinyDB × TinyWebDB), as funções dos blocos lógicos nativos e as diretrizes recomendadas para garantir a integridade e organização das informações em aplicativos móveis.

---

## 📚 Conteúdo da Pesquisa

A pesquisa aborda os seguintes temas:
- Introdução ao MIT App Inventor e suas vantagens para iniciantes
- O componente TinyDB e sua finalidade de persistência local
- Mecanismo de funcionamento (Tags, Valores e Ciclo de Vida dos dados)
- Componentes relacionados e blocos lógicos (StoreValue, GetValue, ClearTag e ClearAll)
- Aplicações práticas no cotidiano e desenvolvimento de jogos
- Análise comparativa: TinyDB × TinyWebDB
- Boas práticas de organização, nomenclatura de tags e gerenciamento de memória
- Conclusão

## 🎯 Objetivo Acadêmico

Este trabalho foi desenvolvido para fins educacionais, com o objetivo de aprofundar os conhecimentos sobre:
- Persistência de dados local em dispositivos móveis
- Lógica de programação visual baseada em blocos
- Estruturas de dados do tipo Chave-Valor
- Diferenciação entre arquiteturas de armazenamento offline e online (nuvem)
- Gerenciamento de memória interna em aplicativos
- Boas práticas de arquitetura de software para dispositivos móveis
